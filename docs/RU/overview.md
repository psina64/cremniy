# Введение в Cremniy

## Назначение

**Cremniy** - это среда разработки для низкоуровневого программирования. Она отличается тем, что объединяет инструменты работы с бинарными файлами, памятью и системным кодом в одном приложении.

IDE предназначена для разработчиков операционных систем, драйверов и системного ПО, а также для тех, кто изучает системное программирование. Cremniy упрощает отладку, анализ памяти и работу с бинарными файлами, позволяя сосредоточиться на логике системы, а не на настройке инструментов.

## Цель

Предоставить удобный инструмент, в котором есть всё необходимое для системного программирования, без необходимости переключаться между разными программами. Cremniy позволяет разработчику сосредоточиться на логике системы и коде, а не на настройке и поиске инструментов.

## Философия

### Всё в одном

Не заставлять разработчика бегать между разными инструментами. **Всё** необходимое для системного программирования должно быть **в одной среде**.

### Минимализм

Инструмент должен быть **максимально простым** в освоении и использовании. **Ничего лишнего**, это касается как GUI так и функционала.
